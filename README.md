# Todo App
Todo App With Golang
## Tech
This App build with golang framework
## Requirement
- Gin Framework
- Gorm

## Installation
```sh
# Download GORM
$ go get gorm.io/gorm
# Download this project
$ go get gorm.io/driver/mysql
# Download Gin Framework
$ go get github.com/gin-gonic/gin
```
